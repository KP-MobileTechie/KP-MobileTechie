# GitHub Metric Cards Redesign — Neon Glass

**Date:** 2026-07-09
**Repo:** KP-MobileTechie/KP-MobileTechie (profile README)
**Status:** Approved design

## Goal

Redesign the two self-hosted GitHub metric SVGs (streak card + contribution
activity graph) from plain/functional to best-in-class "neon glass" UI, while
keeping the existing self-hosted, API-generated, committed-SVG reliability model
unchanged.

## Constraints (hard)

- Output is static SVG embedded in a GitHub README via `<img>` (camo proxy).
- **No JavaScript** — GitHub strips it. Allowed: `<defs>` filters, in-document
  `<style>` CSS (including `@keyframes` and `@media prefers-reduced-motion`),
  and SMIL (`<animate>`, `<animateTransform>`). All of these DO render in an
  `<img>`-embedded SVG (proven by existing animated README widgets).
- No external resources (fonts, images, CSS). System font stack only.
- Generated by `scripts/generate_metrics.py` from the all-time daily
  contribution history already fetched via the GitHub GraphQL API.
- Refresh cadence, workflow, and committed-file approach stay as-is.

## Theme tokens

| Token | Value | Use |
|---|---|---|
| bg | `#06060b` | panel background |
| accent (cyan) | `#22d3ee` | ring, line, total glow |
| fire (pink) | `#f472b6` | flame, peak dot |
| label (purple) | `#a78bfa` | current-streak label, longest glow |
| text | `#e7e9ee` | numbers |
| muted | `#8b8ba7` | labels, dates, axis |
| glass fill | `rgba(255,255,255,.03)` | sub-panels |
| glass stroke | `rgba(255,255,255,.07)` | sub-panel borders |

Shared background per card: solid `bg` + faint radial glow (cyan top-left ~6%,
pink bottom-right ~6%) + subtle dot-grid `<pattern>` (~3% opacity).

## Card 1 — Streak (495×195)

Three frosted-glass sub-panels with gradient dividers that fade at both ends.

- **Left — Total:** big number, soft cyan glow; `TOTAL CONTRIBUTIONS`
  uppercase letter-spaced label; date range (muted).
- **Center — Current Streak:** gradient ring (cyan→purple) drawn 0→360° on
  load, then a subtle infinite glow-pulse; pink glowing flame above the ring;
  big number centered inside; `CURRENT STREAK` label in purple; date range.
- **Right — Longest Streak:** number with purple glow; label; date range.

**Motion:** numbers pop-in (fade + rise + scale), staggered left→center→right;
ring sweep-draw (stroke-dasharray/offset); ring glow-pulse loop ~3s.

## Card 2 — Activity graph (820×260)

- Title `Contribution Graph` top-left; small `peak N/day` chip.
- Glowing cyan line (blurred duplicate underneath = neon halo) over a
  cyan→transparent gradient area fill.
- Month labels on the x-axis; faint horizontal gridlines.
- Pink glowing peak dot at the max day + small value callout; cyan end dot at
  "today".

**Motion:** line draws left→right (stroke-dashoffset), area fades in behind it
after the draw completes; peak dot pulse loop.

## Motion & accessibility

- All animation via CSS `@keyframes` / SMIL. Pseudo "count-up" = fade+rise+scale
  pop (true digit counting needs JS; not available).
- Include `@media (prefers-reduced-motion: reduce)` that disables entrance and
  loop animations (final state shown immediately).

## Data → geometry

- Total, current streak, longest streak (+ date ranges) already computed by
  `compute()` in the generator. No data-layer change.
- Activity graph: last 365 daily counts, scaled to max; peak index → callout.

## Non-goals

- No change to profile-summary-card PNGs or the rest of the Analytics section.
- No new external services or dependencies.
- No change to the fetch/refresh/commit workflow.
